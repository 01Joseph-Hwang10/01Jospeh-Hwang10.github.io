  recommendation_api --> |Recommendation| scripts
  cafe24_api --> |User Data & Mall Data| recommendation_api
  recommendation_api --> |"Banner Manager Scripts
  & Billings"| cafe24_api
  cafe24_api --> |Insert Banner Manager| shop
